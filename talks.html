Elixir Basics Workshop

2016 February 04

A workshop that introduces the basics of Elixir. The repository for this workshop can be found at https://github.com/boulder-elixir/elixir-basics-workshop. As the README explains, there are slides that you can follow to complete the workshop at home.

This workshop was given at Boulder Elixir, a Meetup that I started about Elixir.

Elementary Elixir

2015 December 07

A talk that gives a high level overview of the Elixir language and a brief demonstration of what it's capable of.

The repository for this talk can be found at https://github.com/tracehelms/talk_elementary_elixir.

This talk was given at a Quick Left lunch and learn.

Consuming APIs With Go

2015 May 05

As my first sizable project in Go, I thought I would share my experiences on consuming an API with Go. This is a command line app that finds a random restaurant via Yelp's API.

The repository for the project can be found at https://github.com/tracehelms/golunch.

This talk was given at the Boulder Gophers Meetup group.
